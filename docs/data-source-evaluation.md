# 免费行情数据源评估

> 状态：已确认  
> 调研日期：2026-07-23

## 产品所需数据

- 上交所、深交所、北交所当前上市 A 股清单。
- 六位股票代码对应的当前股票名称。
- 指定日期的不复权日线开盘价和收盘价。
- 每周约 1 至 2 次、每次不超过 10 只股票。

## 推荐方案：AKShare 封装东方财富数据

AKShare 官方股票数据文档提供：

- `stock_zh_a_spot_em()`：一次返回沪深京当前 A 股上市公司的代码和名称，可用于股票范围、上市状态和名称校验。
- `stock_bj_a_spot_em()`：单独返回北交所当前上市 A 股。
- `stock_zh_a_hist(symbol, period="daily", start_date, end_date, adjust="")`：返回沪深京 A 股指定日期范围的日线；`adjust=""` 明确表示不复权，输出包含日期、股票代码、开盘和收盘。

官方资料：

- [AKShare 股票数据文档](https://akshare.akfamily.xyz/data/stock/stock.html)
- [AKShare 特别说明](https://akshare.akfamily.xyz/special.html)

## 推荐调用方式

1. 后端每日首次使用时获取一次沪深京当前 A 股清单，并在内存中缓存至当天结束。
2. 使用清单校验代码、证券类型、当前上市状态并取得股票名称。
3. 对每只股票按指定日期调用日线接口，明确设置 `period="daily"`、`adjust=""`。
4. 开始日期和结束日期相同时只读取该日；不存在记录即按业务规则报错。
5. 数据结构必须由独立 `MarketDataProvider` 适配层输出，Excel 逻辑不得直接依赖 AKShare 字段名。

## 不推荐：搜索引擎和模拟浏览器

不建议通过百度搜索结果或模拟用户打开同花顺、东方财富页面作为主方案：

- 搜索结果不保证提供指定历史日期的开盘、收盘价。
- 页面结构、反爬策略和验证码可能随时变化。
- 文本解析容易把当前价、复权价或其他证券数据误当成目标行情。
- 浏览器自动化速度慢，跨移动平台部署复杂，故障难以定位。

## 风险和边界

- AKShare 是开源数据接口工具，不是交易所官方行情服务，没有可用性保证。
- AKShare 官方声明数据仅供研究参考、不构成投资建议，接口未来可能移除或变化。
- 其东方财富接口属于第三方网页数据来源；产品应保持个人低频使用，避免高频抓取。
- 首版不进行多数据源静默降级，避免不同来源价格口径不一致。接口失败时报告错误并停止生成文件。
- 后续若免费接口不稳定，可以替换适配层，而不改动 Excel 处理流程。

## 最终决策

首版采用 AKShare/东方财富方案，不配置搜索引擎、同花顺页面或东方财富页面模拟操作作为备用路径。数据源失败时终止任务并报告错误。
