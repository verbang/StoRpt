# MVP 验收标准

> 状态：已确认  
> 版本：1.0  
> 更新日期：2026-07-23

本文档将 [product-design.md](product-design.md) 转换为可测试条件。P0 是发布阻断项；任一 P0 未通过，不得发布 MVP。

## 1. 登录、页面与上传

| ID | 优先级 | Given / When / Then |
| --- | --- | --- |
| AC-001 | P0 | Given 未登录或会话失效，When 访问分析、处理、进度或下载接口，Then 返回 `AUTH` 类错误且不接收文件处理。 |
| AC-002 | P0 | Given 正确密码，When 登录成功，Then 通过 `HttpOnly`、`Secure`、`SameSite` Cookie 建立默认 7 天会话，页面不暴露密码或会话值。 |
| AC-003 | P0 | Given 用户选择文件，When 扩展名不是 `.xls` 或 `.xlsx`，或大小超过 10 MB，Then 在处理前报告 `FILE` 错误。 |
| AC-004 | P0 | Given 合法文件，When 完成第一次分析上传，Then 页面显示文件信息、目标工作表和最新时间段，并将开始、结束日期填入输入框；服务端立即删除分析副本。 |
| AC-005 | P1 | Given 桌面或移动浏览器，When 打开页面，Then 必要元素按单列响应式布局可完整使用，移动端无需横向滚动即可完成主流程。 |

## 2. 模板识别与阻断规则

| ID | 优先级 | Given / When / Then |
| --- | --- | --- |
| AC-010 | P0 | Given 工作簿包含恰好一个可见兼容表，When 分析，Then 不依赖工作表名称即可识别该表；其他工作表不参与处理。 |
| AC-011 | P0 | Given 没有候选表、存在多个候选表或候选表隐藏，When 分析，Then 报告 `TEMPLATE` 错误且不生成文件。 |
| AC-012 | P0 | Given 任一时间段标题不符合 `yyyy.MM.dd - yyyy.MM.dd`、日期无效、开始晚于结束、日期重复或开始/结束日期未随区块严格递增，When 校验，Then 阻断处理。 |
| AC-013 | P0 | Given 用户修改最新时间段日期，When 点击处理，Then 再次校验其格式、先后关系及相对上一时间段的严格递增关系。 |
| AC-014 | P0 | Given 最新时间段不是最后业务区块，或其表头以下 A:D 混有公式、合并单元格、备注、汇总或其他业务内容，When 校验，Then 阻断处理。 |
| AC-015 | P0 | Given 文件包含密码、保护、宏、签名、外链、数据连接、透视表、图表、图片、形状或嵌入对象，When 分析，Then 指明不兼容功能并停止，不尝试删除或降级该功能。 |
| AC-016 | P0 | Given 工作簿未启用自动计算或打开时重算，When 分析，Then 判定不兼容，不生成文件。 |

## 3. 股票代码与行情

| ID | 优先级 | Given / When / Then |
| --- | --- | --- |
| AC-020 | P0 | Given 代码由空格、换行、中英文逗号或分号混合分隔，When 解析，Then 得到保持输入顺序的六位代码列表并显示数量。 |
| AC-021 | P0 | Given 代码为空、非六位纯数字、重复、不存在、已退市或不属于沪深京当前上市 A 股，When 校验，Then 整个任务以 `INPUT` 或 `MARKET` 错误失败。 |
| AC-022 | P0 | Given `ST`、`*ST` 或北交所当前上市 A 股，When 校验，Then 作为支持标的正常处理。 |
| AC-023 | P0 | Given 勾选理想进价或理想出价，When 查询行情，Then 分别严格取得开始日不复权开盘价或结束日不复权收盘价，不替换日期、不复权。 |
| AC-024 | P0 | Given 任一股票在所需指定日期无行情，When 查询，Then 汇总相关代码、日期和字段，整个任务失败且不生成部分文件。 |
| AC-025 | P0 | Given 网络超时、连接失败或数据服务临时异常，When 查询，Then 首次失败后最多重试 3 次；确定性错误不重试。 |

## 4. Excel 写入安全边界

| ID | 优先级 | Given / When / Then |
| --- | --- | --- |
| AC-030 | P0 | Given 任意兼容文件，When 处理，Then 唯一允许改变的业务单元格为目标工作表最新时间段标题合并区左上角的 A 列单元格，以及其数据区 A:D；合并范围本身不变，且不得插入或删除整行。 |
| AC-031 | P0 | Given 新代码列表，When 写入，Then A 列从首个数据行连续写入全部代码，保留前导零，并清除该时间段旧 A 列集合中超出新列表长度的尾部代码。 |
| AC-032 | P0 | Given B、C、D 任一复选框未勾选，When 处理，Then 对应列所有原坐标值保持不变，不清空、不移动、不按新代码匹配且不警告。 |
| AC-033 | P0 | Given B、C、D 任一复选框勾选，When 处理，Then 只在本次代码对应连续行写入该列；B 为名称文本，C/D 为行情数值。 |
| AC-034 | P0 | Given 三个复选框均未勾选，When 处理，Then 仍完成代码和日期校验，只更新日期标题与 A 列。 |
| AC-035 | P0 | Given 输入代码数超过现有股票位置，When 扩展，Then 只创建或写入新增行的 A:D 单元格，不触碰 E:S，也不设置整行行高。 |
| AC-036 | P0 | Given 需要新增 A:D 单元格，When 应用格式，Then 只从最后一个可靠标准股票行复制字体、填充、边框、对齐和数字格式；找不到样板行则失败。 |
| AC-037 | P0 | Given 任意成功处理，When 对比输入和输出，Then E:S 的非公式单元格值、公式文本、单元格格式、批注、超链接及合并关系均未被工具改动；所有历史区块和非目标工作表保持不变。公式缓存值按 AC-042 的边界处理。 |

## 5. 输出、自检与失败原子性

| ID | 优先级 | Given / When / Then |
| --- | --- | --- |
| AC-040 | P0 | Given `.xls` 或 `.xlsx` 输入，When 成功，Then 输出保持相同扩展名和工作簿格式，不进行格式互转，源文件不被覆盖。 |
| AC-041 | P0 | Given 成功写入，When 自检，Then 输出可重开，工作表数量、名称、顺序、显示状态不变，日期、A 列、已勾选列及未勾选列分别符合规则。 |
| AC-042 | P0 | Given E:S 含公式，When 自检，Then 不计算或验证收益结果、不写公式缓存，只验证公式文本、格式与合并关系，并保留打开时重算设置。 |
| AC-043 | P0 | Given 任一校验、行情、写入、重开或自检失败，When 任务结束，Then 显示结构化错误和“未生成文件”，不交付临时或部分输出。 |
| AC-044 | P0 | Given 成功任务，When 命名，Then 使用北京时间 `yyyyMMdd.扩展名`；同一浏览器当天后续生成依次使用 `_01`、`_02`，工具不主动覆盖已有文件。 |
| AC-045 | P1 | Given 成功任务，When 浏览器允许自动下载，Then 自动下载；被阻止时显示可用的“下载文件”按钮。 |

## 6. 进度、时限、隐私与部署

| ID | 优先级 | Given / When / Then |
| --- | --- | --- |
| AC-050 | P0 | Given 处理中任务，When 状态变化，Then SSE 按上传、模板校验、代码校验、行情、写入、自检、完成阶段更新进度；失败时停在失败阶段。 |
| AC-051 | P0 | Given 同一账号已有运行中任务，When 再次启动处理，Then 拒绝第二个任务且不创建并发 Excel 操作。 |
| AC-052 | P0 | Given 任务运行达到 180 秒仍未完成，When 超时，Then 终止任务、报告 `SYSTEM` 错误并不生成文件。 |
| AC-053 | P0 | Given 正常完成或失败，When 清理，Then 正常立即删除任务临时文件，异常文件最迟 10 分钟删除，系统不提供历史下载。 |
| AC-054 | P0 | Given 任意请求和日志，When 检查缓存头与日志内容，Then Excel、行情、认证和下载响应禁止缓存；日志不含文件内容、交易数据、密码、会话、完整文件名或原始接口响应。 |
| AC-055 | P1 | Given 单镜像部署，When 在普通 Linux Docker 主机启动，Then 前端、FastAPI、JRE 和 Excel Worker 可在 HTTPS 反向代理后完成端到端流程。 |

## 7. 发布测试矩阵

发布前至少覆盖：

- `.xls` 与 `.xlsx` 各一套正常样本、动态扩展样本、代码缩减样本和不兼容功能样本。
- 三个复选框的 8 种组合，验证未勾选列逐单元格保持不变。
- 周末、休市、停牌、无效代码、重复代码、退市或非 A 股、行情网络失败和 180 秒超时。
- Windows/macOS/Linux 的主流 Chromium 浏览器，以及 iOS Safari、Android/HarmonyOS 主流浏览器的关键流程。
- 保存前后工作簿结构化差异检查，重点确认 E:S、历史区块和非目标工作表没有越界变化。
