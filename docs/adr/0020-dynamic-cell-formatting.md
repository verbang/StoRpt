# ADR-0020：新增股票行只复制 A 至 D 单元格样式

- 状态：已接受
- 日期：2026-07-23

## 背景

股票数量可能超过模板当前已有数据行。工具不能插入整行或修改 E 至 S，但新增的 A 至 D 单元格仍需要与模板视觉和数据类型保持一致。

## 决策

- 已存在的目标单元格只写允许的值，保留其原格式。
- 新增数据行的 A 至 D 样式复制自最新时间段最后一个标准股票数据行。
- 复制范围仅包括字体、填充、边框、对齐和数字格式。
- 不复制原值、公式、批注、超链接或整行属性。
- A 列使用文本格式；B 列写文本；C、D 写数值并保留行情源精度。
- 不设置整行行高，不复制或修改 E 至 S。
- 无法找到可靠样板行时，文件不兼容并停止处理。

## 影响

- 新增行只有 A 至 D 具备模板样式，E 至 S 保持原状或空白。
- Excel 处理组件需要支持单元格级样式复制，不能使用整行复制。
