# ADR-0012：通过结构唯一识别目标工作表

- 状态：已接受
- 日期：2026-07-23

## 背景

样表的工作表名称为 `Sheet1`，但名称可能被用户修改，工作簿也可能包含其他工作表。工具必须可靠确定唯一处理目标，同时保护其他工作表。

## 决策

- 工具扫描工作簿中的所有可见工作表，不依赖固定名称。
- 候选工作表必须包含可解析的时间段标题、A 至 D 固定业务表头，并通过时间段严格递增校验。
- 候选工作表数量必须恰好为一个。
- 没有候选时报告“不存在符合模板的工作表”；存在多个候选时报告“存在多个候选工作表，无法确定目标”。
- 两种错误均停止处理，不生成文件。
- 隐藏工作表不能成为目标工作表。
- 除唯一目标工作表外，其他可见或隐藏工作表不参与业务处理并保持原样。

## 影响

- 用户可以修改目标工作表名称，不影响识别。
- 多个相同结构的投资组合工作表暂不支持在一次任务中选择处理。
- 模板校验需要输出候选工作表名称及失败原因，便于用户定位问题。
