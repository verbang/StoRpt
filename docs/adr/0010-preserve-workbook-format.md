# ADR-0010：输出文件保持源工作簿格式

- 状态：已接受
- 日期：2026-07-23

## 背景

把 `.xlsx` 转换为 `.xls` 需要重新表达整个工作簿，可能改变公式缓存、样式内部结构或不被目标格式支持的功能，与保护原文件格式的目标冲突。

## 决策

- `.xls` 输入输出 `.xls`。
- `.xlsx` 输入输出 `.xlsx`。
- 工具不执行 `.xls` 与 `.xlsx` 之间的格式转换。
- 除本次明确授权的单元格操作外，不随意修改原工作簿的公式、样式、合并关系和其他结构。
- 输出前后必须校验受保护区域及关键工作簿结构保持一致；失败时不交付输出文件。

## 影响

- 原始需求中“统一另存为 `.xls`”被本决策替代。
- 输出文件扩展名随输入格式变化。
- `.xls` 最大 65,536 行、`.xlsx` 最大 1,048,576 行，这些客观限制决定单个文件可容纳的最大股票数量。
- Excel 处理组件必须分别可靠支持两种格式的读写，不能通过中间格式转换实现 `.xls` 支持。
