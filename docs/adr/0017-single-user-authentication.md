# ADR-0017：使用单用户密码保护公网工具

- 状态：已接受
- 日期：2026-07-23

## 背景

工具以网页/PWA 形式部署并包含文件上传、免费行情查询能力。即使仅供个人使用，公网无认证接口也可能被他人滥用。

## 决策

- 使用一个由部署环境变量配置的访问密码。
- 不提供用户名、注册、找回密码和用户数据库。
- 服务端使用安全密码哈希进行验证，不保存明文密码。
- 登录成功后使用带 `HttpOnly`、`Secure`、`SameSite` 属性的 Cookie 建立会话，默认有效期 7 天。
- 文件上传、行情查询和处理接口全部要求认证。
- 连续登录失败达到阈值后执行临时锁定。
- 日志不得记录密码、会话令牌或认证 Cookie。

## 影响

- 部署时必须设置强访问密码和会话签名密钥。
- 忘记密码时通过修改部署环境变量重置，不提供在线找回流程。
- 无需开发用户管理后台。
