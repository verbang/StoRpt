# ADR-0015：首版使用 AKShare 东方财富行情

- 状态：已接受
- 日期：2026-07-23

## 背景

产品需要免费获得沪深京当前上市 A 股代码、名称及指定日期的不复权开盘价和收盘价。使用场景为个人低频，允许免费接口偶发不可用。

## 决策

- 首版后端使用 AKShare 接入东方财富数据。
- 使用 `stock_zh_a_spot_em()` 获取沪深京当前上市 A 股清单、代码和名称。
- 使用 `stock_zh_a_hist()` 获取历史日线，显式设置 `period="daily"`、`adjust=""`。
- 股票清单按自然日缓存，历史价格按任务和指定日期查询。
- 不使用百度搜索或模拟浏览器访问同花顺、东方财富网页。
- 不进行多数据源静默降级；接口失败时终止任务并报告错误。
- 通过 `MarketDataProvider` 适配层隔离 AKShare 字段和调用方式。

## 影响

- 产品依赖第三方免费接口，不提供行情服务等级保证。
- AKShare 或东方财富接口发生变化时，需要更新适配层。
- 数据只用于个人研究参考，不构成投资建议。
- 错误信息应区分网络失败、接口变化、空数据和指定日期无行情。
