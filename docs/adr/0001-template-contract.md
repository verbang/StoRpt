# ADR-0001：以 `platform.xlsx` 兼容结构作为输入契约

- 状态：已接受
- 日期：2026-07-22

## 背景

产品工作将围绕 `platform.xlsx` 模板展开。模板按时间段重复组织数据，并通过公式把股票、理想价格、实际交易数据及收益结果关联起来。

## 决策

工具不作为通用 Excel 编辑器。输入文件必须是 `.xls` 或 `.xlsx`，并且其核心结构与 `platform.xlsx` 兼容。工具识别最新时间段，只修改该区块中本次业务明确允许修改的日期、股票代码、股票名称、理想进价和理想出价字段；其他字段、公式和样式应受保护。

## 影响

- 上传后必须先执行模板兼容性校验，不能仅检查扩展名。
- 产品需要定义可接受的结构差异，例如工作表名称变化、历史区块数量增加或列宽变化。
- 任意结构的 Excel 不在当前产品边界内。
- 模板结构升级时，需要同步维护兼容性版本或迁移规则。
