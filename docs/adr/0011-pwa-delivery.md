# ADR-0011：使用统一网页/PWA 下载交付

- 状态：已接受
- 日期：2026-07-23

## 背景

产品需要覆盖桌面和移动操作系统。浏览器无法普遍获得上传文件的原始本地路径，也不能静默写入原目录。

## 决策

- 所有目标平台统一使用网页/PWA。
- 输出文件通过浏览器或操作系统文件流程下载。
- 不保证输出文件保存到上传文件同目录。
- 输出文件使用北京时间处理当天的 `yyyyMMdd` 作为基础文件名，并保持输入扩展名。
- 产品不得主动覆盖用户已有文件。
- 当前浏览器保存当天的应用生成计数：首次使用基础名称，之后追加 `_01`、`_02` 并依次递增。
- 更换浏览器、设备或清除浏览器数据后计数可以重新开始；若本地仍有同名文件，接受浏览器再次重命名。

## 影响

- 保存位置由浏览器、操作系统设置或用户选择决定。
- 网页无法读取下载目录，因此无法可靠得知本地是否已经存在同名文件。
- `_01`、`_02` 只反映当前浏览器记录的应用生成次数，不等同于下载目录中的真实同名文件序号。
