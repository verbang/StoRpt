# ADR-0008：可选字段采用独立复选和保留原值语义

- 状态：已接受
- 日期：2026-07-23

## 背景

股票名称、理想进价和理想出价可以独立决定是否填入。先前“单选框”表述可能被误解为三个选项互斥，同时需要明确未选择字段是清空还是保留。

## 决策

- 股票名称、理想进价和理想出价使用三个互相独立的复选框。
- 三个复选框默认均未勾选，允许选择零项、一项、多项或全部。
- A 列股票代码始终按本次输入列表更新。
- B、C、D 只有对应复选框勾选时才允许写入。
- 未勾选字段不写入、不清空，保留原单元格值。
- 原值按单元格坐标保留，不按股票代码匹配、搬移或复制。
- A 列代码变化或尾部代码被清除时，同一行未勾选的 B、C、D 仍保持原值。
- 三项均未勾选时，仍执行代码范围校验，并仅处理日期和 A 列代码。

## 影响

- 处理逻辑必须建立列级写入许可，不能先清空 A:D 再回填选中字段。
- 未勾选字段可能与新代码不对应；这是保留原值规则的直接结果，工具不应擅自修正。
- 工具不提示未勾选字段与当前股票代码之间的对应关系。
- 自检不判断未勾选字段与新股票代码是否在业务上匹配，但必须确认这些字段与处理前同坐标单元格值一致，以证明工具没有越权清空或移动原值。
