# ADR-0007：仅支持三家交易所当前上市 A 股

- 状态：已接受
- 日期：2026-07-23

## 背景

六位证券代码可能对应股票、基金、债券等多种证券，单纯依据代码格式或前缀无法可靠确认其属于中国 A 股。产品需要明确市场、上市状态和输入格式。

## 决策

- 支持上交所、深交所和北交所当前上市的 A 股。
- 支持 `ST`、`*ST` 等仍处于上市状态的 A 股。
- 不支持已退市股票、B 股、基金、ETF、指数、债券、可转债及其他非 A 股证券。
- 用户输入六位纯数字代码；多个代码允许使用空格、换行、英文逗号、中文逗号或分号分隔。
- 解析后保留输入顺序；重复代码视为输入错误，不静默去重。
- 证券类型、所属市场和上市状态必须通过当前证券清单校验，不得只依赖代码前缀推断。
- 任一代码无效时，整个任务停止，不回填、不生成文件。

## 影响

- 行情数据能力必须包含北交所证券清单和历史日线。
- 证券清单需要定期更新，并记录更新时间。
- 已退市股票即使在指定历史日期存在行情，也不属于产品支持范围。
- 错误信息应明确区分格式错误、重复代码、证券不存在、非 A 股和已退市。
