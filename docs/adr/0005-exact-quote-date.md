# ADR-0005：价格查询严格使用指定日期

- 状态：已接受
- 日期：2026-07-23

## 背景

开始日期或结束日期可能遇到周末、休市或个股停牌。产品需要确定是改用邻近行情，还是严格使用用户指定日期。

## 决策

- 理想进价只允许使用开始日期当天的开盘价。
- 理想出价只允许使用结束日期当天的收盘价。
- 不得向前或向后寻找替代交易日。
- 只有用户勾选对应价格字段时，才执行对应日期的行情可用性检查。
- 任意一只股票缺少本次所需行情时，整个任务失败，不进行回填，不生成部分成功文件。

## 影响

- 周末、休市日和停牌日可能直接导致任务失败。
- 错误信息必须包含股票代码、缺失字段和指定日期。
- 多股票任务采用全有或全无的原子处理方式。
