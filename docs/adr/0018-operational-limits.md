# ADR-0018：采用个人低频任务运行限制

- 状态：已接受
- 日期：2026-07-23

## 背景

即使产品仅供个人低频使用，也需要防止异常文件、网络故障或重复点击长期占用免费部署资源。

## 决策

- 单个文件最大 10 MB，每次任务只处理一个文件。
- 同一账号同时只允许一个运行中的任务。
- 单个任务总超时为 180 秒。
- 股票代码不设置业务数量上限，但受文件格式容量和任务超时约束。
- 网络超时、连接失败或服务端临时异常在首次失败后最多重试 3 次，即最多尝试 4 次。
- 指定日期无行情、代码错误、模板不兼容等确定性错误不重试。
- 进度按上传、模板校验、代码校验、行情查询、Excel 写入、自检和完成七个阶段计算。

## 影响

- 前端处理期间禁用重复提交。
- 超时或最终重试失败时不生成部分文件。
- 进度消息需要包含当前阶段，并在可能时标明股票代码和日期。
