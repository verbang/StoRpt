# ADR-0006：使用不复权行情价格

- 状态：已接受
- 日期：2026-07-23

## 背景

历史开盘价和收盘价可能由数据源提供不复权、前复权和后复权等不同口径。产品需要固定口径，避免相同股票和日期因数据源默认参数不同而产生不同结果。

## 决策

- 理想进价使用指定开始日期当天的不复权日线开盘价。
- 理想出价使用指定结束日期当天的不复权日线收盘价。
- 不执行前复权、后复权或其他价格调整。
- 行情适配层必须显式请求或验证不复权口径，不能依赖数据源默认值。

## 影响

- 写入结果对应指定日期市场实际形成的价格。
- 分红、送股和转股可能造成跨日期价格不可直接比较，这是不复权口径的正常表现。
- 数据源验收测试需要用已知样本核对开盘价、收盘价及复权参数。
