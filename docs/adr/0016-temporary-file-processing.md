# ADR-0016：Excel 文件由后端临时处理且不留存

- 状态：已接受
- 日期：2026-07-23

## 背景

浏览器本地处理旧版 `.xls`、公式和样式的保真难度较高。产品需要在文件隐私和可靠 Excel 处理之间确定边界。

## 决策

- Excel 文件通过 HTTPS 上传至后端处理。
- 每个任务使用随机标识和独立临时目录，文件不写入业务数据库。
- 后端完成模板校验、行情查询、回填和自检后直接返回下载响应。
- 下载响应正常结束后立即删除输入、输出临时文件。
- 失败、中断或客户端断开时，临时文件最多保留 10 分钟，由清理任务兜底删除。
- 不提供处理历史和再次下载能力。
- 日志不得包含文件正文、股票交易数据和完整文件名。

## 影响

- 部署环境必须支持 HTTPS、临时磁盘和定时清理。
- 处理流程必须确保异常分支同样执行文件清理。
- 用户关闭下载页面后无法恢复结果，需要重新提交任务。
