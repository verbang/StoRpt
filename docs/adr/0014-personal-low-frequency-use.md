# ADR-0014：产品定位为个人低频工具

- 状态：已接受
- 日期：2026-07-23

## 背景

行情授权、系统部署、并发控制和运维要求取决于产品服务对象及使用频率。

## 决策

- 产品仅供个人使用，不作为公开或商业行情服务。
- 预计每周处理 1 至 2 次，每次约 1 至 2 个 Excel 文件。
- 每个文件通常处理不超过 10 只股票。
- 优先采用免费数据方案，接受无服务等级保证和偶发不可用。

## 影响

- 首版无需账号体系、组织权限、多租户和高并发架构。
- 可以优先评估开源免费行情接口，但仍需遵守其使用声明和来源网站规则。
- 数据源必须可替换，避免免费接口变更导致整个产品无法维护。
