# ADR-0013：采用严格 Excel 功能兼容范围

- 状态：已接受
- 日期：2026-07-23

## 背景

产品要求保持源工作簿格式且不得随意修改受保护内容。不同 Excel 功能在 `.xls`、`.xlsx` 处理组件中的保真程度不同，强行保存可能造成用户不可见的数据损失。

## 决策

- 支持并保留普通单元格值、公式、常规样式、数字格式、合并单元格、行高、列宽、工作表名称及顺序、显示状态、冻结窗格和基础打印设置。
- 加密或需要密码的文件不兼容。
- 启用工作簿结构保护或目标工作表保护的文件不兼容。
- 包含 VBA 宏、数字签名、外部链接、外部数据连接、数据透视表、图表、图片、形状或嵌入文件的文件不兼容。
- 发现当前 Excel 组件无法保证原样保留的其他特殊功能时，同样阻断处理。
- 工具必须报告具体不兼容功能，不得删除或降级该功能后继续生成文件。

## 影响

- `platform.xlsx` 样表属于支持范围。
- 产品是严格模板处理工具，不是通用 Excel 编辑器。
- Excel 组件升级或替换时，可以通过专项保真测试逐步扩大支持范围。
