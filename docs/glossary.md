| 股票代码 | 六位纯数字代码；展示和写入时必须保留前导零 | 已确认 |
| 起始日期 | 用户输入的查询区间开始日期；理想进价必须严格使用该日期 | 已确认不允许替代 |
| 截止日期 | 用户输入的查询区间结束日期；理想出价必须严格使用该日期 | 已确认不允许替代 |
| 指定日期 | 用户明确输入且不得被行情查询替换的开始日期或结束日期 | 已确认 |
| 开盘价 | 股票在指定开始日期当天的不复权日线开盘价格 | 已确认 |
| 收盘价 | 股票在指定结束日期当天的不复权日线收盘价格 | 已确认 |
| 回填 | 在不覆盖源文件的前提下，按列级许可把代码或行情结果写入最新时间段 A 至 D | 已确认 |
| 兼容模板 | 能唯一识别目标工作表、通过时间段与功能校验，并可在不越过写入边界的前提下由 Apache POI 保持原格式的 Excel 文件 | 已确认 |
| 时间段区块 | 由日期标题、A 至 D 固定表头和其下股票数据区域组成的业务区域；样表中的 6 行不是产品容量上限 | 已确认动态延展 |
| 最新时间段 | 全部时间段通过严格递增校验后，位于最下方且开始、结束日期均最大的区块 | 已确认 |
| 理想进价 | 对应股票在开始日期当天的不复权开盘价 | 已确认 |
| 理想出价 | 对应股票在结束日期当天的不复权收盘价 | 已确认 |
| 不复权价格 | 指定日期市场实际形成的日线价格，不对历史分红、送股或转股进行调整 | 已确认 |
| 源 Excel | 用户上传、作为处理输入且不可直接覆盖的兼容模板文件 | 支持 `.xls`、`.xlsx` |
| 输出 Excel | 系统完成回填后生成且扩展名、工作簿格式与源文件一致的新文件 | 已确认保持原格式 |
| 受保护列 | 最新时间段中的 E 至 S 列以及所有历史区块；工具不得对其执行任何写操作 | 已确认硬约束 |
| 动态数据区 | 从最新时间段表头下一行开始、由本次股票代码数量决定长度的连续 A 至 D 区域；其下方不得存在其他业务内容 | 已确认 |
| 代码完整替换 | 清除最新时间段原有 A 列代码，再以本次输入列表形成新的 A 列代码集合 | 已确认 |
| 按字段写入 | B、C、D 分别由独立复选框控制；勾选才写入，未勾选则不写也不清空 | 已确认 |
| 按单元格保留 | 未勾选的 B、C、D 保持原坐标中的值，不随 A 列股票代码重新匹配或移动 | 已确认 |
| 不兼容格式 | 时间段标题无法解析、日期无效、日期重复或区块日期未按位置严格递增的输入文件 | 阻断处理且不生成文件 |
| 当前上市 | 在任务处理时仍处于上市状态；已退市证券不在支持范围 | 已确认 |
| 原格式输出 | `.xls` 输入输出 `.xls`，`.xlsx` 输入输出 `.xlsx`，不执行格式转换 | 已确认 |
| 网页/PWA 交付 | 所有目标平台均通过浏览器访问，输出文件由浏览器或系统文件流程下载 | 已确认 |
| 基础文件名 | 以北京时间处理当天的 `yyyyMMdd` 命名并沿用输入扩展名 | 已确认 |
| 本地生成计数 | 当前浏览器记录当天由本应用生成文件的次数，用于追加 `_01`、`_02`；不代表下载目录真实文件数量 | 已确认 |
| 目标工作表 | 工作簿中唯一一个可见且同时满足表头、时间段格式和顺序要求的工作表 | 已确认 |
| 候选工作表 | 经结构扫描后可能作为目标工作表的可见工作表；候选数量必须恰好为一 | 已确认 |
| 严格模板兼容 | 只接受能够可靠保留基础单元格、公式、样式、合并关系及基础工作表属性的文件 | 已确认 |
| 不支持的 Excel 功能 | 密码、保护、宏、数字签名、外链、数据连接、透视表、图表、图片、形状、嵌入对象等无法保证原样保留的功能 | 阻断处理 |
| 个人低频使用 | 单一用户每周约处理 1 至 2 次、每个文件通常不超过 10 只股票的非公共服务场景 | 已确认 |
| MarketDataProvider | 向业务层提供当前 A 股清单、股票名称和指定日期不复权价格的统一数据源接口 | 已确认需要适配层 |
| AKShare | 首版免费行情接入库，当前通过东方财富接口提供沪深京 A 股清单和历史日线 | 已确认首版使用 |
| 临时任务目录 | 后端为单次任务创建、与其他任务隔离并在响应完成或超时后删除的文件目录 | 已确认 |
| 无留存处理 | 文件不入库、不保留历史，正常完成立即删除，异常时最长保留 10 分钟 | 已确认 |
| 单用户访问密码 | 不使用用户名或用户数据库、由部署环境变量配置的唯一访问凭据 | 已确认 |
| 可重试错误 | 网络超时、连接失败或服务端临时异常；首次失败后最多重试 3 次 | 已确认 |
| 确定性错误 | 无行情、无效代码、不兼容模板等再次请求不会改变结果的错误 | 不执行重试 |
| 公式缓存值 | Excel 文件中为快速显示保存的公式上次计算结果；工具不读取其业务正确性，也不主动改写 | 不纳入自检 |
| 自动计算 | Excel/WPS 打开工作簿后重新计算公式的工作簿设置 | 输入模板必须启用 |
| 格式样板行 | 最新时间段中用于复制新增 A 至 D 单元格基础样式的最后一个标准股票数据行 | 必须存在 |
| 错误代码 | 由错误类别前缀和编号组成、用于定位失败阶段的稳定标识 | 已确认 |
| 原子失败 | 任意必要校验、查询、写入或自检失败时不交付部分文件 | 已确认 |
| 纯日期 | 不含时分秒和时区、以 `yyyy.MM.dd` 表示的公历日期值 | 已确认 |
| 分析上传 | 文件选择后用于读取模板和默认日期、完成后立即删除的第一次临时上传 | 已确认 |
| 处理上传 | 用户点击处理后重新上传同一文件并正式执行回填的第二次上传 | 已确认 |

| SSE | 服务端向浏览器单向推送任务进度和状态的连接方式 | 已确认 |
